Rohit’s Portfolio

Welcome to the Rohit Connect AI portfolio page. This page showcases production-ready capabilities for Connect AI, AWS deployment, and GitHub-based continuous delivery.

1. Connect

This application is built as a social discovery platform. Key features include:

  • User registration, login, and profile management
  • Real-time nearby search and location-based matching
  • AI-enhanced recommendations and search
  • Messaging and automation support

2. AWS

This project is prepared for AWS deployment using Elastic Beanstalk, Docker, and Kubernetes. The repository includes:

  • Elastic Beanstalk configuration via .elasticbeanstalk
  • Dockerfile and docker-compose.yml for containerized deployment
  • Kubernetes manifests under k8s/
  • Prometheus and Grafana monitoring stack

3. Production

The production setup includes observability, telemetry, and deployment readiness:

  • Prometheus metrics endpoint at /metrics
  • Health and readiness checks at /healthz and /ready
  • OpenTelemetry tracing support
  • Kafka and Redis integration hooks
  • GitHub sync and MLOps helper scripts

How to deploy

To deploy the latest GitHub changes to AWS, the recommended flow is:

  1. Push code to GitHub
  2. Use GitHub Actions or AWS CodePipeline
  3. Deploy to Elastic Beanstalk or ECS
  4. Monitor the app with Grafana / Prometheus

Live services

Once deployed, the key URLs are:

  • Application: http://connectsingai.com/rohitportfolio
  • Metrics: /metrics
  • Readiness: /ready
  • Grafana dashboard: /grafana (if configured)
